## Quality Gates (Objective, Not Vibe-Based)

- Immersion clarity: users can explain scenario stakes, actors, and current objective without guessing.
- Branch consequence: at least one decision materially changes downstream pressure/context.
- Mentor usefulness: mentor output is context-aware and actionable, not generic template phrasing.
- Evaluation credibility: scoring signal ties to rubric behavior, not superficial proxies.
- Stability and trust: no blocking UX confusion points in primary persona journeys.

## Execution Sequence (Today -> Tomorrow+)

- If all gates pass by tomorrow, publish go/no-go with launch-ready confidence.
- If gates do not pass, do not force finalization; publish no-go + 24-48h extension sprint with concrete acceptance retest.
